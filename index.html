¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML y se utiliza para estructurar y presentar contenido en la web; sirve para organizar textos, imágenes, enlaces y listas, además de permitir integrar de forma nativa elementos multimedia como audio,video y gráficos sin necesidad de programas externos, lo que facilita la creación de aplicaciones interactivas junto con CSS y JavaScript; es importante porque constituye la base del desarrollo web moderno, mejora la semántica con etiquetas como

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title>Mi página</title>
</head>
<body>
</body>
</html>

!DOCTYPE html → Indica que el documento usa HTML5. html lang="es" → Raíz del documento, con atributo lang para el idioma. head → Contiene metadatos (no visibles directamente). meta charset="UTF-8"> → Define la codificación de caracteres. title → Título que aparece en la pestaña del navegador. body → Contiene todo el contenido visible.

La etiqueta

HTML5 semántico

header

La etiqueta

se utiliza para definir el encabezado de una página o sección, normalmente contiene el logotipo, el título y la barra de navegación principal.

nav

La etiqueta

se utiliza para definir la sección de navegación de una página web, normalmente contiene enlaces que permiten al usuario moverse entre las distintas secciones o páginas del sitio.

main

La etiqueta

se utiliza para definir el contenido principal de la página, es decir, la información más importante y única que no se repite en otras secciones como encabezados o menús.

section

La etiqueta

se utiliza para dividir el contenido en bloques temáticos dentro de una página web, agrupando información relacionada en una misma área para mejorar la organización y la lectura.

article

La etiqueta

se utiliza para definir un contenido independiente dentro de la página, como una noticia, entrada de blog o publicación, que puede entenderse por sí mismo sin depender de otras secciones.

footer

La etiqueta

se utiliza para definir el pie de página de una web, donde normalmente se colocan créditos, información de contacto, enlaces secundarios o derechos de autor.

Formularios en HTML5

Los formularios en HTML5 permiten recopilar datos de los usuarios. Se crean con la etiqueta (form) y dentro de ella se usan elementos como (label) para identificar campos, (input) para entradas de texto, correo o números, (textarea) para comentarios largos, (select) para listas desplegables y (button) para enviar la información.

Conclusión

HTML5 es fundamental en el desarrollo web moderno porque introduce etiquetas semánticas que mejoran la organización del contenido, facilita el uso de multimedia sin complementos externos y ofrece mayor accesibilidad. Gracias a sus nuevas funciones, los sitios web son más rápidos, compatibles con distintos dispositivos y fáciles de mantener, lo que lo convierte en una base esencial para crear páginas actuales y eficientes.
