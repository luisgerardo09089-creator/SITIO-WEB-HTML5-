Aprendiendo HTML5

HTML5 es la quinta y última versión del estándar base de la web, que define etiquetas estructurales y semánticas para crear sitios modernos, accesibles y adaptativos (responsive). Incorpora de forma nativa soporte multimedia (video/audio), API de geolocalización, almacenamiento local y gráficos 2D, reduciendo la dependencia de plugins externos como Flash
